# Interactive E-book Viewer

An immersive, high-end interactive e-book viewer built with React, Tailwind CSS, and Lucide React icons.

## Features

- Cinematic landing page with animated book cover
- Responsive 5x8 paperback book viewer
- Interactive page turning with 3D flip animations
- Chapter navigation with rapid page-flip animation
- Beautiful dark/gold color scheme
- Glassmorphism sidebar navigation

## Getting Started

```bash
npm install
npm start
```

## Built with

- React
- Tailwind CSS
- Lucide React Icons
